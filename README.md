## 🚀 Lancement rapide

```bash
git clone https://github.com/JustineTdrs/portfolio.git
cd portfolio
npm install
npm run dev
```

## 📁 Structure

```
├── .github/workflows/deploy.yml   ← Déploiement auto GitHub Pages
├── src/
│   ├── main.jsx                   ← Point d'entrée
│   └── Portfolio.jsx              ← Composant principal trilingue
├── index.html
├── package.json
└── vite.config.js
```

## 🌐 Déploiement

Automatique via GitHub Actions à chaque push sur `main`.  
Settings → Pages → Source : **GitHub Actions**.
